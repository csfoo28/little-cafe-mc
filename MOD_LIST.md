## Optimization

Mods focused on improving server performance, reducing lag, and optimizing resource usage.

### Performance Mods

- **[FerriteCore](https://modrinth.com/mod/ferrite-core)**  
  Optimizes memory usage for blocks and items, reducing server load.

- **[Lithium](https://modrinth.com/mod/lithium)**  
  General-purpose optimization mod for improving game performance without altering vanilla mechanics.

- **[Concurrent Chunk Management Engine (C2ME)](https://modrinth.com/mod/c2me-fabric)**  
  Enhances chunk loading and generation for smoother server performance.

- **[ServerCore](https://modrinth.com/mod/servercore)**  
  Improves server tick rates and entity management for better performance.

- **[Krypton](https://modrinth.com/mod/krypton)**  
  Optimizes Minecraft's networking stack to reduce latency and improve throughput.

- **[Let Me Despawn](https://modrinth.com/plugin/lmd)**  
  Allows mobs to despawn faster, reducing server load in high-population areas.

- **[TT20 (TPS Fixer)](https://modrinth.com/mod/tt20)**  
  Dynamically adjusts server settings to maintain stable TPS (ticks per second).

- **[Disconnect Packet Fix](https://modrinth.com/mod/disconnect-packet-fix)**  
  Fixes issues with disconnect packets to prevent client crashes and improve stability.

### Server Management

- **[Chunky](https://modrinth.com/plugin/chunky)**  
  Pre-generates chunks to reduce lag during exploration and improve server performance.

## Structures

Data packs and mods that add new world structures for exploration.

- **[Terralith](https://modrinth.com/datapack/terralith)**  
  Overhauls world generation with diverse biomes and unique structures.

- **[Additional Structures](https://modrinth.com/mod/additional-structures)**  
  Adds a variety of new structures to enhance exploration in vanilla worlds.

## Utilities

Mods and data packs that add quality-of-life features, gameplay enhancements, and server management tools.

### Player Utilities

- **[Teletransportation Accept (TPA)](https://modrinth.com/datapack/tpa)**  
  Adds a teleportation request system for players to travel to each other.

- **[Skin Restorer](https://modrinth.com/mod/skinrestorer)**  
  Allows players to use custom skins on servers, even in offline mode.

- **[Inventory Sorting](https://modrinth.com/mod/inventory-sorting)**  
  Automatically sorts player inventories with a single click.

- **[JourneyMap](https://modrinth.com/mod/journeymap)**  
  Provides a real-time minimap and full-screen map for navigation.

### Gameplay Enhancements

- **[FallingTree](https://modrinth.com/mod/fallingtree)**  
  Enables trees to break in one hit, dropping all logs and leaves.

- **[Dynamic Lights](https://modrinth.com/datapack/dynamic-torches)**  
  Adds portable light sources, like torches, that emit light when held or placed.

- **[Vanilla Refresh](https://modrinth.com/datapack/vanilla-refresh)**  
  Enhances vanilla mechanics with small quality-of-life tweaks.

- **[BeyondEnchant](https://modrinth.com/datapack/beyondenchant)**  
  Expands enchanting with new, balanced enchantments.

- **[Josh's More Foods](https://modrinth.com/datapack/joshs-more-foods)**  
  Adds new food items and recipes to diversify the cooking system.

- **[BlazeandCave's Advancements Pack](https://modrinth.com/datapack/blazeandcaves-advancements-pack)**  
  Introduces hundreds of new advancements for a challenging progression system.

- **[BlazeandCave's Advancements Pack Hardcore Version](https://modrinth.com/datapack/blazeandcaves-advancements-pack-hardcore-version)**  
  A harder version of the advancements pack tailored for hardcore gameplay.

- **[Vanilla Tweaks](https://vanillatweaks.net/picker/datapacks/)**  
  Customizable data packs for quality-of-life features.  
  *Selected Packs*: Back, Homes, Spawn, Multiplayer Sleep.

### Server Utilities

- **[BlueMap](https://modrinth.com/plugin/bluemap)**  
  Generates an interactive 3D web-based map of the server world.

- **[Simple Discord Link](https://modrinth.com/mod/sdlink)**  
  Connects the server to Discord for chat integration and notifications.

## Miscellaneous

Essential APIs and other mods that support the modded ecosystem.

- **[Fabric API](https://modrinth.com/mod/fabric-api)**  
  Core API required for most Fabric mods to function.

- **[Cloth Config API](https://modrinth.com/mod/cloth-config)**  
  Provides a configuration framework for mods to offer customizable settings.

- **[Almanac](https://modrinth.com/mod/almanac)**  
  Adds an in-game guidebook for modded content and server information.

- **[spark](https://modrinth.com/mod/spark)**  
  A performance profiling tool to diagnose and optimize server lag.
